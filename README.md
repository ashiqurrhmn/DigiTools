# DigiTools

## Project Overview

DigiTools is a modern web application that provides multiple useful digital tools in one platform. The project focuses on creating a fast, responsive, and user-friendly experience with clean UI design and interactive functionality.

## Run Locally

Clone the repository

```bash
git clone https://github.com/ashiqurrhmn/DigiTools.git
```

Go to the project directory

```bash
cd DigiTools
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
