Project: DigiTools

Description: DigiTools is a web-based application that provides useful tools for daily digital tasks. Instead of using multiple websites, users can access everything in one platform. It focuses on simplicity, speed, and a smooth user experience.

Technologies Used:
- HTML
- Tailwind CSS, DaisyUI
- React.js
- JavaScript (ES6+)
- React-Toastify (NPM Package)
- JSON (for product data)

Features:
- Multiple Tools in One Place
- Clean & User-Friendly UI
- Fast & Responsive

Live site: https://digi-tools-plum.vercel.app/
